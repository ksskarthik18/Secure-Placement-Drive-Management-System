How Digital Signatures Work

Signing (HR)

  1. Create data string: student_id:company:package
  2. Hash it with SHA-256
  3. Sign hash with RSA private key
  4. Store signature with result

Verification

  1. Recreate data string from database
  2. Hash it with SHA-256
  3. Verify signature with RSA public key
  4. If mismatch: DATA TAMPERED!
